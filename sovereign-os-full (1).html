Deployment & Activation

Step-by-step guide to deploy RootWise Sovereign OS to production.

Step 1: Environment Setup

# Clone repository
git clone https://github.com/rootwise/sovereign-os.git
cd rootwise-sovereign-os

# Install dependencies
pnpm install

# Configure environment
cp .env.example .env
# Edit .env with:
# - DATABASE_URL (PostgreSQL connection)
# - REDIS_URL (Redis connection)
# - OPENAI_API_KEY (LLM integration)
# - JWT_SECRET (Session signing)

Step 2: Database Setup

# Create PostgreSQL database
createdb rootwise_sovereign

# Run migrations
pnpm db:migrate

# Seed initial data (chambers, agents, axioms)
pnpm db:seed

# Enable pgvector extension
psql rootwise_sovereign -c "CREATE EXTENSION IF NOT EXISTS vector"

Step 3: Deploy to Production

# Option A: Vercel (Recommended)
vercel deploy --prod

# Option B: Railway
railway up --environment production

# Option C: Self-hosted
docker build -t rootwise-sovereign .
docker run -p 3000:3000 rootwise-sovereign
